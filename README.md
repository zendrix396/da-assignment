# Product Placement Analysis

## Overview

This project analyzes **retail sales performance** to understand how **product placement, pricing, discounts, and foot traffic** influence sales.

Using the **Sample Superstore dataset**, we built an **interactive Tableau dashboard** that provides insights into category performance, customer segments, price competition, and sales trends.

The goal of this analysis is to help retailers **optimize product placement and pricing strategies** to improve overall sales performance.

---

## Dataset

The analysis uses the **Sample Superstore dataset**, which includes retail transaction data such as:

* Product Category and Sub-Category
* Sales and Profit
* Customer Segment
* Order and Ship Dates
* Product Price
* Competitor Price
* Discount Applied
* Store Foot Traffic

The dataset helps evaluate how different factors influence sales outcomes.

## Key Visualizations

The Tableau dashboard contains the following visualizations:

1. **Sales by Category**
   Compares average sales across Furniture, Office Supplies, and Technology.

2. **Sub-Category Analysis**
   Shows detailed performance within each product category.

3. **Sales Trend Over Time**
   Line chart analyzing how sales change across years.

4. **Competitor Price vs Product Price**
   Scatter plot analyzing pricing competitiveness.

5. **Sales by Customer Segment**
   Visualizes how Consumer, Corporate, and Home Office segments contribute to sales.

6. **Foot Traffic vs Sales**
   Examines the relationship between store traffic and sales performance.

7. **Discount vs Sales Impact**
   Analyzes how discount levels affect sales.

8. **Category Price Distribution**
   Box plot showing price variation across categories.

## Insights

Some key insights derived from the dashboard include:

* Technology products tend to generate higher average sales.
* Discount levels significantly influence sales behavior.
* Higher foot traffic correlates with increased sales activity.
* Customer segment analysis helps identify high-value buyers.
